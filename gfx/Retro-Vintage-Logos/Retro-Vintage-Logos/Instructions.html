Customising the logos

Simply clicking a text element with the Type tool will allow you to edit the wording. Make sure you have the correct fonts installed!

Each logo has been built using scalable vector shapes and elements. With the exception of the texture layer mask the entire design can be resized within the PSD document without loss of quality. Unlink the Layer Mask from the Group to allow the scalable elements to resize independently.
